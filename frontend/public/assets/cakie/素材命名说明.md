# Team CAKIE 素材命名说明

## 统一规则

- 所有蛋糕主题图片统一放在 `frontend/public/assets/cakie/`。
- 页面引用路径统一以 `/assets/cakie/` 开头。
- 文件名使用“中文用途说明_英文稳定标识.png”的格式，方便设计同学查找，也方便代码检索。
- 插画、头像和图标建议导出为 PNG，并保留透明背景。
- 页面引用素材时必须提供加载失败 fallback。图片不存在时使用 `.cakie-placeholder` 或页面原有占位内容，不应出现破图图标或报错。
- 不创建空白 PNG 占位文件。正式素材到位后，直接使用下表中的文件名放入本目录即可。

## 页面 fallback 约定

推荐图片组件保留加载失败状态，并在失败时渲染：

```jsx
{imageError ? (
  <div className="cakie-placeholder">素材准备中</div>
) : (
  <img src="/assets/cakie/素材文件名.png" alt="素材说明" onError={() => setImageError(true)} />
)}
```

对于小尺寸导航或功能图标，图片缺失时可继续显示原有文字标签或简洁占位内容，避免影响导航和业务操作。
