## 1. 产品概述

### 1.1 产品定位

AI 统筹组长 (AI Calendar) 是一款面向**大学生团队**的智能协作平台。核心理念是让 **AI 担任组长角色**——用户组建团队后，AI 自动分解项目任务、按成员技能智能分配工作、在群聊中以自然语言提供实时指导，同时提供 AI 私人助手进行个人任务管理。

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| 主要用户 | 大学生团队、创赛队伍、课程小组 |
| 核心场景 | 课程大作业、竞赛项目、跨学科团队合作 |
| 痛点 | 团队分工不合理、沟通成本高、不知如何拆解大型项目、缺乏进度追踪 |
| 用户特征 | 熟悉手机操作、习惯聊天式交互、需要零门槛的引导式体验 |

### 1.3 核心亮点

| 亮点 | 说明 |
|------|------|
| AI 即组长 | AI 自动拆任务、按技能分工、实时跟进，成员只需确认和执行 |
| 自然语言驱动 | 群聊 @ai 即可提问，AI 理解项目上下文后智能回复 |
| AI 联网搜索 | DeepSeek Function Calling + DuckDuckGo，AI 可实时搜索互联网获取竞品/市场信息 |
| 人在回路 | AI 分配的任务必须经成员确认/打回，确保人类决策权 |
| 文件智能解析 | 上传 PDF/Word/PPT/图片，AI 自动提取内容辅助理解需求 |
| AI 私人助手 | 转发群名片获取个人化任务指导，支持一键创建任务 |

## 4. 完整交互流程

### Step 1: 注册与技能录入
用户注册时选择专业方向和擅长技能，形成个人技能名片，为后续 AI 智能分配提供数据基础。

### Step 2: 组建团队
创建群组 → 通过邀请码或好友列表邀请成员入群。

### Step 3: 启动 AI 工作流（人在回路）
群主点击「人齐了，开始吧！」→ 输入项目需求描述（支持上传 PDF/Word/PPT/图片附件）→ AI 根据需求+成员技能自动分解任务并分配。

### Step 4: 任务确认（人工决策）
每位成员收到分配的任务后，可选择「接受」或「打回」。打回会触发 AI 重新分配。所有成员确认后进入执行阶段。

### Step 5: 执行与协作
- 看板追踪任务进度（待办→进行中→已完成）
- 群聊中 @ai 随时提问获取指导（支持联网搜索竞品/市场信息）
- AI 私聊页可转发群名片获取个人化建议
- 讨论页 AI 自动联网搜索参考案例和灵感

### Step 6: 数据追踪
用户统计页展示完成率、连续打卡天数、能力画像等数据。
