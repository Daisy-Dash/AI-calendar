## 1. 产品概述

### 1.1 产品定位

AI 统筹组长 (AI Calendar) 是一款面向**大学生团队**的智能协作平台。核心理念是让 **AI 担任组长角色**——用户组建团队后，AI 自动解释项目方向、搜索参考案例、根据组员讨论方案拆分任务、按成员技能智能分配、设置提交节点、追踪进度，并在群聊中以自然语言提供实时指导。

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| 主要用户 | 大学生团队、创赛队伍、课程小组 |
| 核心场景 | 课程大作业、竞赛项目、跨学科团队合作 |
| 痛点 | 团队分工不合理、沟通成本高、不知如何拆解大型项目、缺乏进度追踪 |
| 用户特征 | 熟悉手机操作、习惯聊天式交互、需要零门槛的引导式体验 |

### 1.3 核心亮点

| 亮点 | 说明 |
|------|------|
| AI 即组长 | AI 解释任务/搜案例/拆任务/分工/拆节点/评进度，覆盖全流程 |
| 自然语言驱动 | 群聊 @ai 即可提问，AI 理解项目上下文后智能回复 |
| AI 联网搜索 | DeepSeek Function Calling + DuckDuckGo/Tavily/Bing 三引擎，按4角度拆分搜索词，智能过滤+相关性评分，搜索结果以可点击卡片展示，全组共享持久化 |
| 三层任务结构 | 项目 → 大任务（按技能分人） → 提交节点（带 DDL + 提交要求） |
| 凭证驱动进度 | 上传文件 → AI 评估 → 自动更新进度条 → 同步群聊 |
| 人在回路 | AI 分配必须经过成员确认/打回；方案讨论由人主导 |
| 任务专属 AI 聊天 | 每个任务有独立 AI 助手，支持联网搜索/案例/拆分/凭证 |

## 4. 完整 6 阶段工作流

### 状态机
```
gathering → discussing → confirming → in_progress → completed
```

### Step 1：注册与技能录入（gathering）
邮箱+昵称+密码注册 → 选择头像/专业/技能/工具 → 形成个人技能名片。

### Step 2：组建团队
创建群组 → 自动生成 6 位邀请码 → 拉好友（搜索好友列表）或分享群号让对方搜索加入。

### Step 3：启动工作流（gathering → discussing）
群主点击「人齐了，开始吧！」→ 输入项目需求描述（可上传 PDF/Word/PPT/图片附件）→ **AI 做两件事**：
1. **任务理解 + 方向建议 + 关键提问**（消息流展示）
2. **联网搜索参考案例**（顶部固定栏展示卡片，可点击外链）

**此阶段不创建任务**，群组状态变为 `discussing`，等待小组讨论。

### Step 4：组员讨论 → 提交方案（discussing → confirming）
顶部出现"💭 小组讨论中…/📝 提交方案"入口 → 任意成员可点击提交方案：
- 文本框写讨论后的方向/调整/各人想做的部分
- 提交后 AI 删除旧的待确认任务，**根据方案重新拆 3-6 个大任务并按技能分配到人**，发送任务卡片消息

### Step 5：任务确认（confirming）
每位成员的"待确认"任务自动弹出**居中浮窗**：
- 显示任务标题/描述/优先级/提交节点
- 异议输入框
- "打回重分" / "确认接受" 按钮
- **接受时后端自动调用 AI 拆 3-5 个提交节点**（含"要提交什么 + 渐进 DDL"）

### Step 6：执行阶段（in_progress）
所有人确认后状态变为 `in_progress`：

- 首页项目卡下显示我的任务列表（按 DDL 升序，可折叠）
- 群聊"📋 任务清单"看团队全部任务（自己的可点击）
- 点击任务 → **TaskChatPage（任务专属 AI 聊天）**：
  - 进度条上叠加节点圆点 + 下方时间戳
  - 点击节点圆点 → 详情浮窗（标题/状态/DDL/要提交什么/进度/上传凭证按钮）
  - 快捷操作：🔍 联网搜索、📎 上传凭证、📋 拆分任务、💡 找案例、📅 执行计划
- **凭证上传 → AI 评估进度 → 自动更新进度条 → 同步群聊（凭证文件 + 进度通知）**
- 群聊也可上传节点凭证（"📌 我的任务"面板里每个任务有 📎 凭证按钮）

### Step 7：逾期与提醒
- 任务卡片逾期 → 红色边框 + `animate-pulse` 闪烁
- 进入 TaskChatPage → 自动弹"⏰ 任务已逾期"提示
- 进入首页 → 自动弹"任务逾期"汇总弹窗

### Step 8：知识库 + 群聊 AI
- 群聊"📚 知识库"面板查看所有上传文件
- 每个文件有"💡 问 AI"按钮 → AI 给该文件的核心要点 + 项目应用建议
- 群聊 @ai + 任意问题 → AI 结合任务/成员/进度回复

### Step 9：单聊 AI 助手
- AI 私聊页支持"💌 转发群名片"
- 选择一个群 → AI 读取你在该群的任务/进度/DDL → 给个人化指导
- 支持深度分析（提取任务）+ 一键批量创建
