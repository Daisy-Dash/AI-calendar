> 版本: 2.0 | 日期: 2026-06-06

## 1. 设计理念

### 1.1 核心设计原则

| 原则 | 说明 | 体现 |
|------|------|------|
| **温暖亲切** | 降低工具感，让用户感到舒适 | 手绘风格、暖色系 |
| **AI 克制** | AI 是引导者而非替代者 | 先调研后确认、不直接出成稿 |
| **人在回路** | 关键决策必须由人类确认 | 授权大按钮、手动认领 |
| **渐进引导** | 从陌生到掌控的全流程陪伴 | 6步线性流程、小白指南 |
| **零门槛** | 不需要额外学习成本 | 聊天式交互、标签化选择 |

### 1.2 视觉关键词

**手绘** · **暖色调** · **圆润** · **轻盈** · **团队感**

## 4. 自定义组件设计

### 4.1 任务卡片 `<TaskCard />`（核心自定义组件）

```
┌─────────────────────────────────┐
│ [难度标签]    [技能标签1][标签2]  │
│                                  │
│ 📋 任务标题                      │
│ 任务简要描述...                  │
│                                  │
│ 👤 认领人: xxx    📅 预计3天     │
├─────────────────────────────────┤
│ ← 左滑删除   长按编辑 →          │
└─────────────────────────────────┘
```

特性:
- 不对称圆角模拟手绘
- 侧滑手势显示删除按钮
- 长按震动反馈+编辑模式
- 拖拽支持（在泳道间移动）
- 根据状态自动变色边框

### 4.2 技能标签 `<SkillTag />`

```css
padding: 6px 14px;
border-radius: 16px;
background: #FFF5E6;
border: 1.5px solid #FFD6A5;
/* 选中态 */
background: linear-gradient(135deg, #FF9F43, #FF8C00);
color: white;
```

### 4.3 灵感卡片 `<InspirationCard />`

横向滑动展示，卡片包含:
- 竞品标题
- 简要描述
- 类型标签（APP/网站/论文等）

### 4.4 AI 对话气泡

```css
/* AI 气泡 — 左侧 */
background: linear-gradient(135deg, #FFF5E6, #FFE8CC);
border-radius: 16px 16px 16px 4px;

/* 用户气泡 — 右侧 */
background: linear-gradient(135deg, #FF9F43, #FF8C00);
color: white;
border-radius: 16px 16px 4px 16px;
```

### 4.5 雷达图 `<RadarChart />`

SVG 实现的五维雷达图，展示技能匹配度:
- 编程能力
- 设计能力
- 文案能力
- 调研能力
- 工具熟练度

## 6. 交互设计

### 6.1 必做手势交互

| 手势 | 场景 | 效果 |
|------|------|------|
| 侧滑 | 任务卡片 | 滑出删除按钮 |
| 长按 | 任务卡片 | 进入编辑模式 |

### 6.2 必做动画

| 动画 | 场景 | 效果 |
|------|------|------|
| fadeInUp | 页面转场 | 新页面从底部滑入 |
| 弹跳插入 | 任务拆解生成 | 任务卡片依次弹入看板 |
| 脉冲 | 授权按钮 | 按钮呼吸发光效果 |
| scale | 按钮点击 | 按下缩小+弹回 |

### 6.3 微交互

| 交互 | 效果 |
|------|------|
| 按钮 hover | 轻微上浮 + 阴影加深 |
| 按钮 active | scale(0.95) 按压反馈 |
| 标签选择 | 弹跳选中 + 颜色变化 |
| 认领成功 | 纸屑动画 + 振动 |
| 加载中 | 旋转动画 + 省略号闪烁 |
| 横向滑动 | 灵感卡片惯性滚动 |
