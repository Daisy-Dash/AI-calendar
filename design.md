# AI 统筹组长 — 设计方案文档

> 版本: 3.0 | 日期: 2026-06-08

---

## 1. 设计理念

### 1.1 核心设计原则

| 原则 | 说明 | 体现 |
|------|------|------|
| **温暖亲切** | 莫兰迪低饱和配色，降低工具感 | 奶油色背景、手绘圆角卡片 |
| **AI 原生** | 自然语言是核心交互方式 | 群聊@ai、私聊对话、语音式气泡 |
| **人在回路** | 关键决策必须由人类确认 | 任务确认/打回、工作流启动确认 |
| **移动优先** | 430px 手机壳容器模拟真实App | 底部导航、触控友好、竖屏布局 |
| **零门槛** | 不需额外学习成本 | 聊天式交互、标签化选择、引导提示 |

### 1.2 视觉关键词

**莫兰迪** · **手绘感** · **低饱和** · **圆润** · **温暖** · **奶茶色系**

## 2. 色彩系统 — 莫兰迪手绘风

### 2.1 核心色板

| 色系 | 变量名 | 色值 | 用途 |
|------|--------|------|------|
| Cream 奶油 | cream-50 ~ cream-500 | `#FAF6F1` ~ `#C4B5A4` | 页面背景、卡片底色 |
| Rosa 玫瑰 | rosa-100 ~ rosa-500 | `#F2D5D5` ~ `#B37474` | 主色调、按钮、强调 |
| Sage 鼠尾草 | sage-100 ~ sage-500 | `#D5E2D0` ~ `#6F8F66` | 成功、确认、完成态 |
| Lilac 紫丁香 | lilac-100 ~ lilac-500 | `#E2D8ED` ~ `#9A88B5` | AI功能、辅助色 |
| Dusty 灰蓝 | dusty-100 ~ dusty-500 | `#D4E1EB` ~ `#7A9AB0` | 信息提示、进行中 |
| Choco 巧克力 | choco-100 ~ choco-500 | `#C4B5A4` ~ `#4D4038` | 文字、边框 |
| Caramel 焦糖 | caramel-100 ~ caramel-500 | `#E8D5B7` ~ `#B08F60` | 强调、高亮 |

### 2.2 功能色

| 场景 | 颜色 | 对应色系 |
|------|------|----------|
| 主按钮/CTA | rosa 渐变 | rosa-300 → rosa-400 |
| 成功/完成 | sage-300 | `#A8BFA0` |
| AI 相关 | lilac-200 | `#C8BDD7` |
| 信息/进行中 | dusty-300 | `#9FB5C4` |
| 警告/紧急 | 柔和红 | `#D4A5A5` → `#B37474` |
| 文字主色 | choco-500 | `#4D4038` |
| 文字辅助 | choco-200 | `#A89888` |

### 2.3 状态颜色

| 任务状态 | 颜色 | 含义 |
|---------|------|------|
| 待办 | cream-300 | 等待处理 |
| 进行中 | dusty-300 | 正在执行 |
| 已完成 | sage-300 | 任务完成 |
| 待确认 | caramel-200 | 等待成员确认 |

```
╭─────────────────────────────────╮
│  📋 任务标题              🔴 高  │
│  任务描述文字...                 │
│                                 │
│  👤 负责人    📅 2026-06-15     │
```

## 6. 交互设计

### 6.1 手势交互

| 手势 | 场景 | 效果 |
|------|------|------|
| 侧滑 | 任务卡片 | 滑出删除/编辑按钮 |
| 下拉 | 消息列表 | 加载更多历史消息 |
| 长按 | 群名片 | 弹出转发到AI选项 |

### 6.2 动画系统

| 动画 | 场景 | 效果 |
|------|------|------|
| fadeInUp | 页面转场 | 新页面从底部滑入 |
| 弹跳插入 | 任务创建/AI生成 | 卡片依次弹入 |
| 脉冲呼吸 | 「人齐了」按钮 | 按钮发光吸引点击 |
| scale | 按钮点击 | 按下缩小+弹回 |
| 打字机 | AI 回复 | 文字逐字出现效果 |
| 旋转 | 加载中 | emoji旋转等待态 |

### 6.3 微交互

| 交互 | 效果 |
|------|------|
| 按钮 hover | 轻微上浮 + 阴影加深 |
| 按钮 active | scale(0.95) 按压反馈 |
| 标签选择 | 弹跳选中 + 颜色渐变 |
| 发送消息 | 气泡从右下滑入 |
| 任务确认 | 绿色闪烁 + 对勾动画 |
| 任务打回 | 轻微抖动 + 橙色高亮 |
| 文件上传 | 进度圈 + 完成打勾 |
