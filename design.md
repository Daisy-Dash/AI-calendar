# AI 统筹组长 — 设计方案文档

> 版本: 4.3 | 日期: 2026-06-13

---

## 1. 设计理念

### 1.1 核心设计原则

| 原则 | 说明 | 体现 |
|------|------|------|
| **温暖亲切** | 莫兰迪低饱和配色，降低工具感 | 奶油色背景、手绘圆角卡片 |
| **AI 原生** | 自然语言是核心交互 | 群聊 @ai、任务专属 AI 聊天 |
| **人在回路** | 关键决策由人确认 | 任务确认/打回、方案讨论、凭证上传 |
| **移动优先** | 430px 手机壳容器 | 底部导航、触控友好 |
| **零门槛** | 不需额外学习成本 | 聊天式交互、引导提示 |
| **层级清晰** | 三层任务结构 | 项目 → 大任务 → 节点（圆点） |

### 1.2 视觉关键词

**莫兰迪** · **手绘感** · **低饱和** · **圆润** · **温暖** · **奶茶色系** · **节点感**

## 2. 色彩系统 — 莫兰迪手绘风

### 2.1 核心色板

| 色系 | 变量 | 色值 | 用途 |
|------|------|------|------|
| Cream 奶油 | cream-50 ~ cream-500 | `#FAF6F1` ~ `#C4B5A4` | 页面背景、卡片底色 |
| Rosa 玫瑰 | rosa-100 ~ rosa-500 | `#F2D5D5` ~ `#B37474` | 主色、按钮、强调 |
| Sage 鼠尾草 | sage-100 ~ sage-500 | `#D5E2D0` ~ `#6F8F66` | 成功、完成 |
| Lilac 紫丁香 | lilac-100 ~ lilac-500 | `#E2D8ED` ~ `#9A88B5` | AI 相关、辅助 |
| Dusty 灰蓝 | dusty-100 ~ dusty-500 | `#D4E1EB` ~ `#7A9AB0` | 信息、进行中 |
| Choco 巧克力 | choco-100 ~ choco-500 | `#C4B5A4` ~ `#4D4038` | 文字、边框 |
| Caramel 焦糖 | caramel-100 ~ caramel-500 | `#E8D5B7` ~ `#B08F60` | 强调高亮 |

### 2.2 功能色

| 场景 | 颜色 |
|------|------|
| 主按钮/CTA | rosa-300 → rosa-400 渐变 |
| 成功/完成 | sage-300 |
| AI 相关 | lilac-200 |
| 信息/进行中 | dusty-300 |
| 警告/紧急/逾期 | red-400 + animate-pulse |
| 文字主色 | choco-500 |
| 文字辅助 | choco-200 |

### 2.3 任务状态色

| 状态 | 颜色 | 含义 |
|------|------|------|
| 待办 | cream-300 | 等待处理 |
| 待确认 | caramel-200 | AI 已分配，等用户确认 |

| 已打回 | rosa-300 | 用户拒绝，等待重分 |
| 已逾期 | red-400 | DDL 过 + pulse 闪烁 |

### 2.4 群组 6 阶段状态色

| 状态 | Emoji | 颜色 |
|------|:---:|------|
| gathering 召集中 | 📢 | cream-300 |

## 6. 交互动效

### 6.1 页面转场
- `fade-in-up` 进入动画（30 → 0 px translateY + opacity）

### 6.2 按钮反馈
- `active:scale-[0.98]` 点击缩放
- `hover:shadow-md` 鼠标悬停

### 6.3 节点圆点
- `hover:scale-125` 放大
- `active:scale-110` 点击
- 逾期：`animate-pulse`

### 6.4 任务卡片
- 逾期：`animate-pulse` 闪烁
- 进度条过渡：`transition-all duration-500`

### 6.5 折叠展开
- `chevron rotate-180` 旋转
- 内容 `fade-in-up` 进入

### 6.6 AI 加载
- 三个圆点弹跳（错峰 animationDelay：0ms / 150ms / 300ms）

### 6.7 联网搜索
- 顶部 banner 带跳动小点 + "AI 正在联网搜索参考案例中..."

## 7. 浮窗设计规范

### 7.1 统一规格

| 属性 | 值 |
|------|------|
| 蒙层背景 | `bg-transparent`（完全透明） |
| 蒙层 z-index | `z-[200]` |
| 浮窗背景 | 纯白 `bg-white` |
| 浮窗圆角 | `rounded-3xl` |
| 浮窗阴影 | `shadow-2xl` |
| 浮窗宽度 | `max-w-[500px] w-full`（mobile 撑满 - 24px 边距，桌面封顶 500px）|
| 浮窗高度 | `max-h-[85vh]` 或 `max-h-[88vh]` |
| 内部布局 | flex-col：固定头部 + 可滚动内容 + 固定底部 |

### 7.2 蒙层限制 CSS

```css
.app-container .fixed.inset-0 {
  max-width: 540px;    /* 比浮窗多 40px 留 padding */
  width: 100%;
  left: 50%;
  right: auto;
  transform: translateX(-50%);
}
```

### 7.3 关闭方式

- ✕ 按钮
- 点击蒙层
- ESC 键（未实现）

```css
}
```

### 8.2 触控优化

- 最小点击区 44×44px
- 按钮间距 ≥ 8px
- 弹出菜单底部安全区 `pb-8`

### 8.3 横向滑动

- AI 案例卡片：`overflow-x-auto scrollbar-hide`
- 节点时间戳：4 个均分（justify-between）
